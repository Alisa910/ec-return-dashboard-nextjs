## 🔄 如何更新数据

### 方式1：更新真实数据后重新转换

1. **替换原始数据**
   ```
   将新数据复制到：
   public/data/EC退货率变化_原始.csv
   ```

2. **运行转换命令**
   ```bash
   npm run convert-data
   ```

3. **刷新浏览器**（Ctrl + R）

### 方式2：直接修改标准格式数据

如果你能直接导出标准格式，跳过转换步骤：

1. 将数据导出为以下格式：
   ```csv
   Year,Channel,Shop_Name,Return_Rate_YTD,Net_Sales
   2024,天猫旗舰店,MM 天猫旗舰店,46,434633487.91
   2025,天猫旗舰店,MM 天猫旗舰店,49,463188665.53
   ```

2. 保存为：`public/data/EC退货率变化.csv`

3. 刷新浏览器
