# ⭐ EC退货率Dashboard - 快速开始指南

> 3分钟快速启动您的Dashboard

## 📋 开始前检查清单

- [ ] 已安装 Node.js (v18+)
- [ ] 已安装 Python 3.x
- [ ] CSV数据文件已准备好
- [ ] 已安装pandas库 (`pip install pandas`)

## 🚀 三步启动

### 第一步：安装依赖 📦

**双击运行**：`📦安装依赖.bat`

或手动执行：
```bash
npm install
```

### 第二步：处理数据 📊

**双击运行**：`📊处理数据.bat`

或手动执行：
```bash
python scripts/convertData.py
```

### 第三步：启动项目 🚀

**双击运行**：`🚀一键启动.bat`

或手动执行：
```bash
npm run dev
```

浏览器访问：http://localhost:3000

## ✅ 验证成功

如果看到以下界面，说明启动成功：

- ✅ 页面显示"EC 退货率分析仪表盘"标题
- ✅ 显示多个渠道卡片（TM、JD、DY等）
- ✅ 卡片显示销售YOY和退货率YOY数据
- ✅ 点击卡片可进入详情页

## 🎯 快速导航

### 数据更新

1. 替换CSV文件（保持路径不变）
2. 双击运行 `📊处理数据.bat`
3. 刷新浏览器页面

### 推送到GitHub

1. 双击运行 `🌐推送GitHub.bat`
2. 输入GitHub仓库地址
3. 输入提交信息（可选）
4. 等待推送完成

### 部署到Vercel

1. 访问 https://vercel.com
2. 使用GitHub登录
3. 导入您的仓库
4. 点击Deploy
5. 获得生产环境URL

## ⚠️ 常见启动问题

### 问题1：Node.js版本过低
**错误信息**：`Error: Requires Node.js >= 18`
**解决方案**：下载最新版Node.js → https://nodejs.org/

### 问题2：找不到Python
**错误信息**：`python不是内部或外部命令`
**解决方案**：
1. 下载Python → https://www.python.org/
2. 安装时勾选"Add Python to PATH"

### 问题3：找不到CSV文件
**错误信息**：`未找到数据文件`
**解决方案**：
1. 确认文件路径：`D:\HQ文件夹data\EC退货率\EC退货率变化.csv`
2. 或修改 `scripts/convertData.py` 中的 `CSV_PATH`

### 问题4：缺少pandas库
**错误信息**：`ModuleNotFoundError: No module named 'pandas'`
**解决方案**：
```bash
pip install pandas
```

### 问题5：端口被占用
**错误信息**：`Port 3000 is already in use`
**解决方案**：
1. 关闭占用端口的程序
2. 或修改端口：`npm run dev -- -p 3001`
